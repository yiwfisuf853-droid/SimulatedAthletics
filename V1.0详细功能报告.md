# 模拟竞技游戏 V1.0 详细功能报告

## 当前范围

- 当前版本聚焦单人模式、排行榜、本地复盘、快捷键与基础主题切换。
- 页面默认以英文为主，同时保留中英文切换能力。
- 所有文案统一收敛到 `src/i18n/messages.ts` 维护。
- 旧版左侧设置栏与独立拓展配置条已废弃。

## 1. 核心玩法

### 1.1 已确认

- 玩家需要持续点击当前全局最小数字所在的单元格。
- 懒人模式会高亮当前最小数字所在位置。
- 最后一击仍保留 1 秒等待后结算的规则。
- 得分受连击、点击间隔与懒人模式惩罚系数影响。
- 本地排行榜会自动保存已完成成绩，并支持点击进入复盘。

### 1.2 当前 UI 基线

- 棋盘始终是视觉中心。
- 宽屏时键盘谱位于棋盘左侧，虚拟键盘位于棋盘下方。
- 计时器在任何场景均可见，作为独立区域与计分区分离。
- 屏幕宽度不足时，排行榜可以隐藏，优先保证棋盘显示。

## 2. 游戏设置

- 常规边长支持 `2x2` 到 `6x6`。
- 拓展模式开启后，顶部边长切换为 `6/8/10/12/16/20`。
- 拓展模式下固定层数锁定为 `1`，其他层数按钮置灰不可选。
- 支持种子输入，便于生成可复现局面。
- 支持显示 / 隐藏排行榜、虚拟键盘、键盘谱与坐标提示。

## 3. 键盘谱与复盘

- 键盘谱的目标是帮助用户快速识别按键并定位。
- 展示时保持完整，不做折叠或动态裁剪。
- 单键模式强调当前主键，弱化双键存在感。
- 虚拟键盘是独立于键盘谱的底部辅助区域。
- 复盘支持播放、暂停、速度切换、单步前进与单步后退。
