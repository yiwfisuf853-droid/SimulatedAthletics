# 模拟竞技

这是一个基于 `React + TypeScript + Vite + Zustand` 的模拟竞技原型项目，当前重点是单人模式、键鼠协同、本地排行榜与复盘体验。

## 当前设计基线

- 默认 UI 以英文为主，语言映射统一维护在 `src/i18n/messages.ts`。
- 语言切换入口放在“更多”面板中，当前支持英文与中文。
- 新版没有独立左侧设置栏，也没有单独的拓展配置条。
- 常用设置放在顶部，“更多”面板承载低频设置；拓展模式开关也放在“更多”里。
- 打开拓展模式后，顶部边长按钮从 `2/3/4/5/6` 切换为 `6/8/10/12/16/20`。
- 拓展模式下固定层数锁定为 `1`，其他层数按钮置灰不可选。
- 对局与复盘时，棋盘始终是视觉中心。
- 键盘谱在宽屏布局中放在棋盘左侧，虚拟键盘始终放在棋盘下方。
- 计时器需在任何场景可见，且与计分栏分离显示。

## 本地运行

```bash
npm install
npm run dev
```

## 常用脚本

```bash
npm run dev
npm run build
npm run lint
```
